Portfolio

Below are some projects I've recently worked on. More to come!

VinylDNS

Sep 2018 - Present

For Drexel’s CCI senior project, myself and a group of 6 other students are helping to redesign the front-end of Comcast’s open source project Vinyl DNS. VinylDNS is a vendor agnostic front-end for enabling self-service DNS and streamlining DNS operations.
Note: This project spans the full academic year, and therefore is still in progress. The final presentation will be in June 2019. You can go to the project website here for updates.

Apple Music Webplayer

Sep 2018 - Present

For one of my interface design classes, the term project was to redesign the interface of a website with Adobe XD. I took an alternative route with this project, as my site choice technically didn’t really redesign an existing website. Instead, I proposed the creation of an Apple Music webplayer. The goal was to revamp the standard iTunes layout and deliver the experience of Apple Music’s mobile app. You can play with the prototype here.

"B Movie" Poster

Sep 2018 - Present

For my digital design tools class, our final assignment was to create a “B Movie” poster featuring ourselves as the main subject. Using Adobe Illustrator, I created “The Night of the Living Jerm”, an ode to B horror movies. While this was the final submission for the course, I still plan on continuing to update this to make it as “cinematic” as possible.
